# Szavazat számláló

Cél a `useEffect` használatának a gyakorlása

Egy kedvelés számlálót kell készíteni. A szavazó felület szürke, amíg nem kattintunk rá. Like-ra kattintáskor a szimbólum zöld színű lesz, a dislike marad szürke a számláló eggyel nő. Dislike-ra kattintva a szimbólum piros lesz, a like marad szürke. A Két ikon között a számláló csökken, de nem lehet kevesebb nullánál.

Az ikonokat célszerű a [Font Awesome](https://fontawesome.com/) készletből venni.
