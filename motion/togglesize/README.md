# React + [Framer Motion](https://www.framer.com/) Examples

Animációkkal kapcsolatban az alábbi web technológiákat használhatjuk:

## Natív web technológiák

### CSS Animations & Transitions

Az alapok – egyszerű, hatékony, nincs függőség. Keyframe animációkra, hover effektekre, átmenetekre tökéletes. Ha a logika nem bonyolult, gyakran ez a legjobb választás.

### Canvas API (2D)

Pixelalapú rajzolás JavaScripttel. Nagy teljesítményű, testreszabható, de nem vektoros – skálázáskor romolhat a minőség.

### SVG + SMIL / CSS animáció

Vektorgrafika, ami skálázható és DOM-alapú, így JavaScripttel könnyen manipulálható. UI ikonokhoz, illusztrációkhoz, adatvizualizációhoz ideális.

### WebGL GPU-gyorsított 3D (és 2D) grafika

Nagyon nagy teljesítmény, de közvetlenül bonyolult használni – ezért inkább könyvtárakon keresztül érdemes elérni.

## JavaScript könyvtárak

### [Framer Motion](https://www.framer.com/motion/)

Egyszerű, de erőteljes animációs könyvtár React-hez. leginkább a komponensek pozíciójának, méretének és stílusának animálására használják, de támogatja a komplex animációkat is, például a layout animációkat és a shared layout animációkat. elsősorban a React közösségben népszerű, de használható más JavaScript környezetekben is.

### [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/)

Nagy teljesítményű, sokoldalú animációs könyvtár, amely nem csak React-hez, hanem bármilyen JavaScript környezethez használható.

### [Anime.js](https://animejs.com/)

Könnyen használható, de erőteljes animációs könyvtár, amely támogatja a CSS, SVG, DOM attribútumok és JavaScript objektumok animációját.

### [Lottie](https://airbnb.design/lottie/)

Adobe After Effects animációk exportálására és megjelenítésére szolgáló könyvtár, amely JSON formátumban tárolja az animációkat, és könnyen integrálható webes és mobil alkalmazásokba.

### [React Transition Group](https://reactcommunity.org/react-transition-group/)

Egyszerű átmenetek létrehozására szolgáló könyvtár React-ben, amely lehetővé teszi a komponensek megjelenésének és eltűnésének animálását.

### [React Reveal](https://www.react-reveal.com/)

Könnyen használható animációs könyvtár React-ben, amely számos előre definiált animációs effektet kínál, például fade, slide, zoom és flip, amelyek egyszerűen alkalmazhatók a komponensekre.

### [React Move](https://react-move.js.org/)

Adatvezérelt animációk létrehozására szolgáló könyvtár React-ben, amely lehetővé teszi a komponensek pozíciójának és stílusának animálását az adatok változásával.

### [React Motion](https://github.com/fisshy/react-motion)

Egyszerű, de erőteljes animációs könyvtár React-hez, amely a fizikai alapú animációkhoz használható.

### [React Flip Move](https://github.com/PrescienceLab/react-flip-move)

Listaelemek animálására szolgáló könyvtár React-ben, amely lehetővé teszi a listaelemek megjelenésének, eltűnésének és átrendezésének animálását.

### [React Spring](https://react-spring.io/)

Fizikán alapuló animációk létrehozására szolgáló könyvtár React-ben, amely lehetővé teszi a komponensek pozíciójának, méretének és stílusának animálását a fizikai törvények alapján.

### [React Pose](https://popmotion.io/pose/)

Egyszerű, de erőteljes animációs könyvtár React-hez, amely lehetővé teszi a komponensek pozíciójának, méretének és stílusának animálását a pose-ok segítségével.

### [Matter.js](https://brm.io/matter-js/)

Fizikai szimulációs könyvtár JavaScript-ben, amely lehetővé teszi a testek közötti ütközések, gravitáció és egyéb fizikai jelenségek szimulálását, és animációk létrehozását ezek alapján.

### [Three.js](https://threejs.org/)

3D grafika létrehozására szolgáló könyvtár JavaScript-ben, amely lehetővé teszi a 3D modellek, animációk és interaktív jelenetek létrehozását a WebGL segítségével.

### [PixiJS](https://pixijs.com/)

2D grafika létrehozására szolgáló könyvtár JavaScript-ben, amely nagy teljesítményű renderelést és animációkat tesz lehetővé a WebGL segítségével.

### [Velocity.js](http://velocityjs.org/)

Gyors és könnyen használható animációs könyvtár JavaScript-ben, amely támogatja a CSS, SVG és DOM attribútumok animációját, valamint a szekvenciális és párhuzamos animációkat.

### [Mo.js](https://mojs.github.io/)

Könnyen használható, de erőteljes animációs könyvtár JavaScript-ben, amely támogatja a komplex animációk létrehozását, például a burst animációkat, a shape morphingot és a timeline animációkat.

### [Popmotion](https://popmotion.io/)

Egyszerű, de erőteljes animációs könyvtár JavaScript-ben, amely támogatja a fizikai alapú animációkat, a keyframe animációkat és a timeline animációkat.

### [Rough.js](https://roughjs.com/)

Kézzel rajzolt stílusú grafika létrehozására szolgáló könyvtár JavaScript-ben, amely lehetővé teszi a vektoros grafika létrehozását egyedi, kézzel rajzolt megjelenéssel.

### [D3.js](https://d3js.org/)

Adatvizualizációs könyvtár JavaScript-ben, amely lehetővé teszi a komplex adatvizualizációk létrehozását, beleértve az animációkat is, amelyek az adatok változásával frissülnek.
